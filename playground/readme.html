Tutlage Playground - In Browser Code Editor

Tutlage Playground is an online in browser code editor quite similar to JsFiddle and JsBin. My idea is not to compete them as they already great in themselves, tutlage playground an open source approach where you download the source files and create your own snippets community using Tutlage Playground.

Version 1.0 is an standalone version which means you have all the flexibility to integrate it with your existing website or blog. It makes use of Php ( 70% or greater ) and Jquery ( 30% of lesser ) to setup a working in browser code editor.

Functionality

Currently it supports web standard languages which includes Html,Css and Javascript libraries

In-built Js Libraries

  1. Plain Javascript
  2. Jquery
  3. MooTools
  4. Prototype
  5. YUI3.6
  6. Dojo
  7. ExtJs

Above are the default libraries which are supported by tutlage playground, you can add more libraries just my adding couple of more lines inside config file.

Adding More Js Libraries

In order to add more js libraries on config.php file and follow the below instructions.

  1. Adding Library Display Name

    Add new library display name inside the following array 

    				$js_array = array(
					'Plain Javascript',
					'Jquery',
					'MooTools',
					'Prototype',
					'YUI3.6',
					'Dojo',
					'ExtJs',
					'Your New Library Display Name'
				);
  2. Adding Library Source File

    Adding new library source file is as easy as adding the display name, just edit the below array as instructed 

    				$js_libs = array(
					'Plain Javascript' => '',
					'Jquery' => '',
					'MooTools' => '',
					'Prototype' => '',
					'YUI3.6' => '',
					'Dojo' => '',
					'ExtJs' => '',
					'Your New Library Display Name' => 'New Library Src Url'
				);
  3. Adding Library OnLoad Function

    Almost all jquery libraries have their onload or ondomready functions, which needs to defined under the following array 

    				$js_onReadyFunctions = array(
					'Plain Javascript' => '** CODE **',
					'Jquery' => "$(function(){** CODE **})",
					'MooTools' => "window.addEvent('domready', function() {** CODE **})",
					'Prototype' => "document.observe('dom:loaded', function() {** CODE **})",
					'YUI3.6' => 'YUI().use("node", function(Y){ Y.on("domready", function(){** CODE **});})',
					'Dojo' => "dojo.addOnLoad(function(){** CODE **})",
					'ExtJs' => "Ext.onReady(function(){** CODE **})",
					'Your New Library Display Name' => 'New Library Dom Ready Function{** CODE **}'
				);

That all is required in order to add new library to tutlage playground, apart from all this it do support external file includes for css and js only files with handy keyword shortcuts to save mouse moving time.

Keyboard Shorcuts

  1. Run Code

    CTRL + S

  2. Download Code

    ALT + D

  3. Reset Playground

    ALT + R

  4. Add External Resources

    ALT + A

FullScreen Support ( Beta ) And Comments

Fullscreen support will toggle the output area to cover the entire width and height of the screen and user can also leave comments which will be appended to the top of the actual file which they will download later.

Getting Playground Values

With all that done to functionality , the key part is the accessibility of the values. Lot of you would be incorporating it to your existing site or blog. That is why i decided to store all the configration values and code values inside sessions. Which will let you access them anywhere on your server without including any dependency files.

		$_SESSION['mashed_code'];  // Complete marked up code
		$_SESSION['js_library'];	// Js Library Source File Url
		$_SESSION['external_libraries']; // List of all included external files ( Will be rendered with the file code )
		$_SESSION['playground_js'];		// Js code written by the end user
		$_SESSION['playground_css'];	// Css code written by the end user
		$_SESSION['playground_html'];	// Html code written by the end user
		$_SESSION['playground_comments'];	// Comments left by the end user
		$_SESSION['selected_js_lib'];	//	Select Js library display name
		$_SESSION['existing_external_libs'];	// List of all included external files ( Will be shown to the user under add files dialog box )
		$_SESSION['external_libs_count'];	// Number of external files included
		$_SESSION['active_tab'];	// Active tab the user was working with
		$_SESSION['js_default_position'];	// Default position to load the js code ( Can be inside head OR before body closing tags )

Setting Playground Values

Setting values is as easier as getting values, all you need to take care of is that they are in the right format which you test by doing an echo on above sessions.

		// It will set the html of the playground to an heading tag with hello world phrase
		$_SESSION['playground_html'] = '<h2> hello world </h2>';

As shown above you can set the value of any session on the page load time and it will be reflected automatically.
